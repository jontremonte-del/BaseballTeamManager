🔥 Firebase Status
Show setup instructions
1
Create a free Firebase project
Go to console.firebase.google.com → Add project → name it diamond-manager → Create.
2
Add a Web App
Click the </> web icon → nickname diamond-manager-web → Register. Copy the firebaseConfig object shown.
3
Enable Realtime Database
Left sidebar → Build → Realtime Database → Create database → Start in test mode → Enable. Copy the database URL.
4
Paste config into code
Open index.html on GitHub. Find FIREBASE_CONFIG near the top and replace each placeholder value.
5
Commit & deploy
Commit in GitHub → Vercel redeploys → sync bar turns green.
⚠️ Test mode expires in 30 days. Go to Realtime Database → Rules and set "read": true, "write": true before expiry.
