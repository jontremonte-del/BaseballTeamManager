🔥 Firebase Setup Instructions

Follow these steps once to enable cloud sync so coaches can view the lineup on any device.

1
Create a free Firebase project
Go to console.firebase.google.com → Add project → name it diamond-manager → Create.
2
Add a Web App
Click the </> web icon → nickname diamond-manager-web → Register. Copy the firebaseConfig object shown.
3
Enable Realtime Database
Left sidebar → Build → Realtime Database → Create database → Start in test mode → Enable. Copy the database URL (https://…firebaseio.com).
4
Paste config into code
Open index.html on GitHub. Find FIREBASE_CONFIG near the top and replace each PASTE_YOUR_…_HERE value. Ensure databaseURL is included from step 3.
5
Commit & deploy
Commit in GitHub → Vercel redeploys in ~30s → sync bar turns green.
6
Share with coaches
Send your Vercel URL. All devices see the same data in real time. No login needed.
⚠️ Test mode expires in 30 days. Before expiry go to Realtime Database → Rules and set "read": true, "write": true.
